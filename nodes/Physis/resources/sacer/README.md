# Módulo SACER (Acopio)

Este directorio controla el flujo físico y logístico de la planta de acopio. Gestiona desde la entrada del camión hasta el análisis de calidad.

## Recursos Disponibles

* **Documentación**:
    * Gestión de **Cartas de Porte (CPG)**.
    * Descarga automatizada de PDFs de CPG.
* **Laboratorio**:
    * Análisis de **Calidad**.
    * Determinación de Variedades y Mermas.
* **Logística**:
    * Gestión de Choferes y Transportistas.
    * Tablas de Tarifas de Flete.
* **Infraestructura**: Administración de Plantas y Silos.

## Uso Común
Automatización de descarga de cartas de porte desde AFIP/Web Services o envío de reportes de calidad a los depositantes.
